## 包说明([Packages](./README.md))  

### backup

备份远程RDB和AOF到本地.  

### extension

命令扩展和Module扩展.  

### failover

PSYNC2 故障转移示例.  

### file

解析AOF, RDB文件, 合并分割AOF, RDB文件, 过滤RDB文件.  

### huge

通过socket或文件处理大KV.  

### migration

在两个Redis之间做数据迁移.  

### other

广播事件与定时同步.  

### socket

通过socket同步数据.  

### stream

Redis-5.0 stream 示例.  

### util

实用工具.
